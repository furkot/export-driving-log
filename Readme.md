# @furkot/export-driving-log

Generate CSV driving log from [Furkot] trip data.

## Install

```sh
$ npm install --save @furkot/export-driving-log
```

## Usage

```js
import drivingLog from '@furkot/export-driving-log';

drivingLog(trip);
```
